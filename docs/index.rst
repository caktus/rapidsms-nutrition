Welcome to rapidsms-nutrition's documentation!
==============================================

rapidsms-nutrition is an app which monitors child growth using `RapidSMS
<http://rapidsms.readthedocs.org>`_.

It allows you to create instant statistical assessments based on WHO Child
Growth Standards, using the `pygrowup <https://github.com/ewheeler/pygrowup>`_
library.
